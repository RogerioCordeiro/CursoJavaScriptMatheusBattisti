### Aula 327 - REST e RESTful

- REST é uma documentação para padronizar as apis;
- RESTful seguinifica que uma api utiliza os principios do REST;

### Aula 328 - Verbos HTTP

- GET - solicita dados;
- POST - inserir dados;
- DELETE - apagar dados;
- PUT - atualizar dados;

### Aula 329 - O que é Express
